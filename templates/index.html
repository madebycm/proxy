Certificate Installation

  1. Make sure the proxy is running
  2. Open Safari and navigate to http://mitm.it
  3. Click on the Apple icon to download the CA certificate
  4. Double-click the downloaded certificate
  5. In Keychain Access, find the mitmproxy certificate
  6. Double-click it and set to "Always Trust"
  7. Close and save the changes
